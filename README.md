express-api-es6
===============

Dependencies
------------
[NodeJS and NPM](https://nodejs.org/en/download/)

[PM2](http://pm2.keymetrics.io/)

[GULP](http://gulpjs.com/)

Installation
------------
Run this command to install all project dependencies.

`npm install`

Run
---
`npm run start`

Test an endpoint: `curl -X GET http://localhost:9000/categories`

Test
---
`npm run test`
