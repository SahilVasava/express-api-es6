express-api-es6
===============

# Dependencies

[NodeJS and NPM](https://nodejs.org/en/download/)
[PM2](http://pm2.keymetrics.io/)
[GULP](http://gulpjs.com/)

# Installation

`npm install`

# Run

`npm run start`

Test an endpoint
`curl -X GET http://localhost:9000/categories`

# Tests

## Test Unit

`npm run test`

## Test Integration

`npm run test:integration`
